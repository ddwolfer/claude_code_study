# 03 · Claude Code 實戰示範

這一課以**影片示範**為主,直接展示 [[claude-code]] 的核心工具如何在真實開發情境中運作。文字說明很少,重點在「看著做、跟著試」。

---

## Claude Code 的內建工具箱

[[claude-code]] 預裝了一整套開發工具,涵蓋日常任務的各個面向:

| 類別 | 代表工具 | 說明 |
|---|---|---|
| 檔案讀寫 | Read、Write、Edit | 讀取與修改專案檔案 |
| 指令執行 | Bash | 在 shell 執行任意命令 |
| 目錄管理 | Glob、LS | 搜尋與列舉檔案樹 |
| 程式碼理解 | Grep | 搜尋符號、跨檔追蹤 |

---

## 工具的智慧組合

[[claude-code]] 真正強大之處不在單一工具,而是能**自動把多個工具串接**,處理複雜的多步驟問題。

```mermaid
flowchart LR
    A["使用者輸入需求"] --> B["Claude 拆解步驟"]
    B --> C["Read 讀取相關檔案"]
    C --> D["Grep 定位符號"]
    D --> E["Edit 修改程式碼"]
    E --> F["Bash 執行測試"]
    F --> G{"測試通過?"}
    G -- 是 --> H["回報結果"]
    G -- 否 --> C
```

**關鍵洞察**:面對一個「幫我修這個 bug」的請求,[[claude-code]] 會自行決定要先讀哪些檔、要搜尋哪些符號、要修哪些地方,最後驗證結果 — 使用者不需要逐步指揮。

---

## 本課重點在動手

這課主要是影片操作示範。最佳學習方式:

1. 打開終端機,進入任何一個有程式碼的專案
2. 執行 `claude` 啟動互動模式
3. 試著提一個真實問題(例:「幫我找出這個函式所有呼叫點」)
4. 觀察 [[claude-code]] 如何選用並串接工具
